Piano Ear Trainer Web App

The Piano Ear Trainer is designed to be a fun tool to help users improve their sense of pitch. At the press of a button a user is played a random piano note. A user can then input their guess of where on the piano that note came from. Alternatively the aspiring musician can also improve their sheet music reading skills by generating a random note on the staff. The app keeps track of the users score and displays this as a percentage in a scss graphic dial.

Tune-bot tuning calculator

The popular tune-bot tuning calculator is a JavaScript based calculator that takes inputs such as drum size, number of drums, and tuning preference and provides the user with suggested tunings based on their individual setup.

Android Bass Guitar Tuner

This is an android app that generates a random note from a bass guitar. The user is presented with a UI that resembles a bass guitar's fret board. The user will guess where on the fret board the note came from by pressing on the screen. The user will then be told if their guess was right or wrong.

Rails Menu Ordering System

This is a project done in rails intended to allow restaurant patrons to order food through their mobile devices. Restaurant owners can upload their full menu, while customers can place menus through the app that will then be displayed and processed in the kitchen. Additionally the order information can be used for analytics purposes.

Chinatown Review site

A group project done in rails for featuring and reviewing restaurants in Boston's Chinatown neighborhood. Users can create new restaurants, upload photos, write reviews, and vote on reviews as well.

Link Shortener

Too Short Link Shortener is a url link shortener. Similarly to products like bitly the link shortener allows users to input long urls and returns you a shorter url which will redirect to the originally provided url. Additionally a user can automatically generate html links by providing a destination url and label.
